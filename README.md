### iTerm Bindings

Profiles --> Keys

- Send Escape Sequence: `[H` --> ⌘←
- Send Escape Sequence: `[F`--> ⌘→
- Send Escape Sequence: `f` --> ⌥→
- Send Escape Sequence `b` --> ⌥←
- Send HexCodes `0x17` --> ⌥+delete
- Send HexCodes `0x15` --> ⌘+delete
